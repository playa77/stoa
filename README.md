# Canonical Agent Workbench

A human-directed platform for high-fidelity compression, analysis, deliberation, and action across information and real workspaces.

## Status

Under construction. Not yet usable.

## What This Is

A single-operator cognitive workbench that combines research synthesis, multi-perspective deliberation, workspace operations, and evaluation infrastructure into one coherent system. The human remains the operator of record. Agents are instrumented machinery with bounded authority, traceable actions, and measurable outcomes.

## Architecture

Five layers: interaction, orchestration, capability, protocol, evaluation — with traces, storage, and skills cutting across all of them. Provider-agnostic. Skills-based internal control (Claude Skills standard). Integrated evaluation spine for measurable improvement.
